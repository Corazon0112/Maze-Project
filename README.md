# Maze Game
This is a simple maze game built using Java Swing's JPanel and graphics libraries. The game features a maze that can switch between 2D and 3D views and the player must navigate through to reach the end.

## Features:
The program uses JPanel to generate a maze from a txt file where "#" are walls, "E" is the starting point, and "F" is the finish line.
The user can press the space bar to switch between 2D and 3D views.
The player can turn and move through the maze using the arrow keys.
The game includes obstacles such as walls, which must be navigated around, and also includes tools like a flashlight and sprays.
The game keeps track of the player's total steps and total turns and displays it on the screen.

## Technology Stack:
Java,
Swing JPanel and graphics libraries

## Getting Started:
To get started with the Maze Game, simply clone the repository and run the game using a Java IDE such as Eclipse or IntelliJ. The game can also be run from the command line by compiling the source code and running the resulting class file.
